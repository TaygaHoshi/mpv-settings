# mpv-settings
my settings for MPV (Linux). I use a desktop computer with Intel CPU + AMD GPU.

### LINUX INSTALLATION
```bash
mv ~/.config/mpv ~/mpv-old
git clone https://github.com/TaygaHoshi/mpv-settings.git ~/.config/mpv
```
